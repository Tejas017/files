8. Generative models (deep dive)

Core categories:

Autoregressive: model p(x)=∏ p(x_t|x_<t). Examples: n-gram, RNN LM, Transformer decoders (GPT). Trained with teacher forcing using cross-entropy. Good likelihoods. Slow sampling sequentially.

Latent variable models (VAE):

Generative model pθ(x|z) p(z). Evidence p(x)=∫ pθ(x|z)p(z) dz. Intractable.

ELBO: log pθ(x) ≥ E_{qφ(z|x)}[log pθ(x|z)] - KL(qφ(z|x) || p(z)).

Reparameterization: z = μ + σ ⊙ ε, ε ~ N(0,I) to backprop through sampling.

Loss = reconstruction loss + KL. Produces smooth latent space.

Generative Adversarial Networks (GANs):

Minimax: min_G max_D V(D,G) = E_x[log D(x)] + E_z[log(1-D(G(z)))].

Training alternates discriminator and generator updates.

Issues: mode collapse, instability. Fixes: WGAN (earth mover distance), gradient penalty, spectral norm, label smoothing, two-time-scale updates.

Normalizing Flows:

Invertible transform x = f(z). Change of variables: p_X(x)=p_Z(z)|det ∂f^{-1}/∂x|.

RealNVP, Glow are examples.

Score-based / Diffusion models (DDPM / Score matching):

Forward: add noise in T steps to transform data into Gaussian.

Train a model to predict noise εθ(x_t,t). Objective simplifies to E[||ε - εθ(x_t,t)||^2].

Sampling: iterative denoising from noise using learned model. High-quality image generation.

Evaluation metrics:

Images: FID (Fréchet Inception Distance), IS.

Text: Perplexity, BLEU, ROUGE. For LLMs also human eval and factuality checks.

Likelihood-based vs sample-quality trade-offs.

Mathematical highlights (compact):

VAE ELBO derivation sketch:

log p(x) = ELBO + KL(q(z|x)||p(z|x))
ELBO = E_q[log p(x|z)] - KL(q(z|x)||p(z))


GAN objective and gradient updates alternate D and G.

Code sketch: VAE encoder/decoder (PyTorch)

class Encoder(nn.Module):
  def __init__(...): ...
  def forward(self,x):
    h = self.net(x)
    mu, logvar = h.chunk(2, dim=1)
    return mu, logvar

def reparam(mu, logvar):
  eps = torch.randn_like(mu)
  return mu + eps * (0.5*logvar).exp()

9. Large Language Models & practical GenAI

Pretraining objectives: autoregressive LM (GPT) vs masked LM (BERT).

Scaling laws: more data + params → better performance (rough rule).

Fine-tuning: supervised fine-tuning, instruction tuning, RLHF (collect preferences, train reward model, PPO).
